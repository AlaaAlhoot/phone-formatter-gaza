==============================
   منسق أرقام الجوال
   تطوير: علاء الحوت
==============================

التشغيل المباشر:
----------------
1. تأكد من تثبيت Python 3.10+
2. شغّل: pip install -r requirements.txt
3. شغّل: python main.py

تحويل إلى EXE:
--------------
شغّل ملف build.bat بنقرة مزدوجة
الملف التنفيذي سيظهر في مجلد dist/

ملاحظات التحقق من الأرقام:
---------------------------
- يجب أن يكون الرقم 9 خانات
- يجب أن يبدأ بـ 59 أو 56
- يتم حذف الأرقام المكررة تلقائياً
- الأرقام الخاطئة تُصدَّر في ملف Excel منفصل
